# FinalAssignment
my final coding project for this semester of TechBasics, it is called *Sudokenland – The Logic Game*

Sudokenland is a 2D Adventure and logic game which was developed for the Technological Basics Assignment in the second semester at Leuphana university and Hamburg Media School. I tried my best with creating an input/output game that includes story telling. I hope you enjoy playing!

You can find the game on the second branch from this repository.
