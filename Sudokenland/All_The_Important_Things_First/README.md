#  Sudokenland – The Logic Game

**Sudokenland** is a 2D Adventure and logic game which was developed for the Technological Basics Assignment in the second semester at Leuphana university and Hamburg Media School.
 I tried my best with creating an input/output game that includes story telling. I hope you enjoy playing!

-Sarah Kruschinski


You wake up in a chaotic world, where the help of a clever mind is needed. **YOU**! The villain Dyskalkulo hast stolen all the numbers in the world and put them away in a cage.
The queen Legasthenia pleads for your help to restore order.
To free the numbers zero until nine, you have to solve a game of **Sudoku**.
Each level that you finish successfully sets one number free. Just until all of them are freed and complete.

# Requirements

- Python 3.10 or newer
- Pygame 2.x


**Story**

 *"Hero, the land of Sudokenland needs your cleverness."*   

 In a world full of letters but without numbers, chaos reigns:
No clocks
No trade
No order
The villain Dyskalkulo wants a world of letters only and has imprisoned all numbers.
Queen Legasthenia begs you to bring the numbers back.
For every Sudoku level you solve, one number is freed.
Start with Level Zero (Tutorial) and fight your way through all nine levels until order is restored.

*Features*

Character portraits: Queen Legasthenia & Dyskalkulo
10 Sudoku levels (0 = Tutorial, 1–9 regular levels)
A world map with the levels on it
Final scene after Level 9
Word-wrapping story mode (prologue pages, press ENTER to advance)
JSON level system — levels are stored separately and are easy to extend


*Controls*
**Key	Action**
ENTER:	Confirm / Continue
ESC:    Back / Skip / Quit
ARROWS / A,D,W,S:	Navigate menus / Move on the Sudoku board
1–9:	Place digit
0 / BACKSPACE / DEL: Clear cell
H:      gives a hint (up to 3 per level, if enabled)
L:	    Auto-solve the current Sudoku (for testing or help)
E:	    Return to world map
R (title/map):	Reset progress
